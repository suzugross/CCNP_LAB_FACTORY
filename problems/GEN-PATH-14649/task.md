# 問題 GEN-PATH-14649 : OSPF 経路制御（自動生成・冗長トポロジ）

各ルータ間リンクと Loopback には IP 設定済み。

## トポロジ
`RT02` から `RT01` の Loopback へは **`RT04` 経由** と **`RT03` 経由** の 2 つの経路がある（冗長構成）。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **`RT02` が `RT01` の Loopback (`92.92.92.92/32`) へ転送するトラフィックは、必ず `RT04` 経由のみ** とすること（`RT03` 経由は使わせない）。
   - 他ルータ間の到達性は維持すること。

## ループバック
- RT01: 92.92.92.92/32
- RT02: 79.79.79.79/32
- RT03: 16.16.16.16/32
- RT04: 54.54.54.54/32

## リンク
- RT02(10.229.108.1) — RT04(10.229.108.2)  [10.229.108.0/30]
- RT02(10.110.247.1) — RT03(10.110.247.2)  [10.110.247.0/30]
- RT04(10.135.46.1) — RT01(10.135.46.2)  [10.135.46.0/30]
- RT03(10.96.181.1) — RT01(10.96.181.2)  [10.96.181.0/30]
