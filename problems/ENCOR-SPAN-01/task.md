# 問題 ENCOR-SPAN-01 : ローカル SPAN でユーザポートをアナライザへミラー（難易度4）

## シナリオ
アクセススイッチ **SW01** にネットワークアナライザ（IDS）を接続し、特定ユーザポートの
トラフィックをミラーリングして可視化したい。すべて **SW01 1 台**で完結するローカル SPAN を構成する。

| ポート | 用途 |
|--------|------|
| `Gi0/1` | 営業端末（監視対象） |
| `Gi0/2` | サーバ上り（監視対象） |
| `Gi0/3` | **アナライザ接続（ミラー先）** |

- VLAN10(USERS) は設定済み。Gi0/1〜Gi0/3 は VLAN10 のアクセスポート。
- 管理は `Gi3/3` / VLAN999（**触らないこと**）。

## 到達目標 — SW01 のみ
1. **monitor session 1** をローカル SPAN として構成する。
2. **営業端末 `Gi0/1` は送受信の両方向**をミラーする。
3. **サーバ上り `Gi0/2` は受信(rx)のみ**をミラーする（上り方向だけ見たい）。
4. ミラー先(destination)は **アナライザ `Gi0/3`**。

## 制約
- 変更してよいのは **SW01 のみ**。管理 IF(`Gi3/3`)/VLAN999 には触れない。
- source の方向（both / rx）の指定を要件どおりに分けること。

## アクセス
※ このスイッチは SSH 鍵生成タイミング等の都合で、CML コンソールから直接解いてください
（採点は telnet 収集で自動実行します）。

## 採点
```
ansible-playbook playbooks/grade.yml -e problem=ENCOR-SPAN-01 --vault-password-file <(printf 'CCNP\n')
```
※ 採点は `show monitor session 1` の source(方向別)/destination を確認します。
