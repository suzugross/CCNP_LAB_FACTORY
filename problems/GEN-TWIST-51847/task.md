# 問題 GEN-TWIST-51847 : OSPF ルートフィルタ（自動生成・tree・4台）

各ルータ間リンクと Loopback には IP 設定済み。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **RT03** は **RT04 の Loopback (`77.77.77.77/32`)** を自身のルーティングテーブルに**保持しないこと**。
   - 他の全ルータは `77.77.77.77/32` へ到達できること。
   - RT03 自身も、RT04 以外の全 Loopback へは到達できること。

## ループバック
- RT01: 54.54.54.54/32
- RT02: 35.35.35.35/32
- RT03: 73.73.73.73/32
- RT04: 77.77.77.77/32
