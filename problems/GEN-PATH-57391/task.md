# 問題 GEN-PATH-57391 : OSPF 経路制御（自動生成・冗長トポロジ）

各ルータ間リンクと Loopback には IP 設定済み。

## トポロジ
`RT02` から `RT04` の Loopback へは **`RT01` 経由** と **`RT03` 経由** の 2 つの経路がある（冗長構成）。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **`RT02` が `RT04` の Loopback (`62.62.62.62/32`) へ転送するトラフィックは、必ず `RT01` 経由のみ** とすること（`RT03` 経由は使わせない）。
   - 他ルータ間の到達性は維持すること。

## ループバック
- RT01: 81.81.81.81/32
- RT02: 82.82.82.82/32
- RT03: 36.36.36.36/32
- RT04: 62.62.62.62/32

## リンク
- RT02(10.97.131.1) — RT01(10.97.131.2)  [10.97.131.0/30]
- RT02(10.19.222.1) — RT03(10.19.222.2)  [10.19.222.0/30]
- RT01(10.78.47.1) — RT04(10.78.47.2)  [10.78.47.0/30]
- RT03(10.13.160.1) — RT04(10.13.160.2)  [10.13.160.0/30]
