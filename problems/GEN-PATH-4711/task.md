# 問題 GEN-PATH-4711 : OSPF 経路制御（自動生成・冗長トポロジ）

各ルータ間リンクと Loopback には IP 設定済み。

## トポロジ
`RT03` から `RT02` の Loopback へは **`RT01` 経由** と **`RT04` 経由** の 2 つの経路がある（冗長構成）。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **`RT03` が `RT02` の Loopback (`44.44.44.44/32`) へ転送するトラフィックは、必ず `RT01` 経由のみ** とすること（`RT04` 経由は使わせない）。
   - 他ルータ間の到達性は維持すること。

## ループバック
- RT01: 46.46.46.46/32
- RT02: 44.44.44.44/32
- RT03: 86.86.86.86/32
- RT04: 7.7.7.7/32

## リンク
- RT03(10.76.5.1) — RT01(10.76.5.2)  [10.76.5.0/30]
- RT03(10.109.76.1) — RT04(10.109.76.2)  [10.109.76.0/30]
- RT01(10.176.227.1) — RT02(10.176.227.2)  [10.176.227.0/30]
- RT04(10.85.243.1) — RT02(10.85.243.2)  [10.85.243.0/30]
