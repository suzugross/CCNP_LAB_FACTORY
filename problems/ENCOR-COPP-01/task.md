# 問題 ENCOR-COPP-01 : CoPP による制御プレーン保護

## シナリオ
ルータ RT01 が、大量の ICMP によって制御プレーン（ルータ自身の CPU 宛トラフィック）を
圧迫される事象が報告されています。制御プレーンに到達する ICMP を制限して保護してください。

## 到達目標
- RT01 の**制御プレーンに入ってくる ICMP** を **8000 bps** までに制限する。
- 制限内（適合）のトラフィックは通常どおり転送し、**超過分は破棄**する。
- ICMP 以外の制御プレーン宛トラフィックには影響を与えないこと。

## 制約
- 既存のインタフェース／到達性の設定は変更しないこと。
- 制限の対象は ICMP のみとすること。

## 採点
```
ansible-playbook playbooks/grade.yml -e problem=ENCOR-COPP-01 --vault-password-file <(printf 'CCNP\n')
```
