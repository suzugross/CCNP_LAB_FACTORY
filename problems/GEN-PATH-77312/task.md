# 問題 GEN-PATH-77312 : OSPF 経路制御（自動生成・冗長トポロジ）

各ルータ間リンクと Loopback には IP 設定済み。

## トポロジ
`RT03` から `RT01` の Loopback へは **`RT02` 経由** と **`RT04` 経由** の 2 つの経路がある（冗長構成）。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **`RT03` が `RT01` の Loopback (`87.87.87.87/32`) へ転送するトラフィックは、必ず `RT02` 経由のみ** とすること（`RT04` 経由は使わせない）。
   - 他ルータ間の到達性は維持すること。

## ループバック
- RT01: 87.87.87.87/32
- RT02: 83.83.83.83/32
- RT03: 47.47.47.47/32
- RT04: 77.77.77.77/32

## リンク
- RT03(10.115.30.1) — RT02(10.115.30.2)  [10.115.30.0/30]
- RT03(10.99.106.1) — RT04(10.99.106.2)  [10.99.106.0/30]
- RT02(10.149.242.1) — RT01(10.149.242.2)  [10.149.242.0/30]
- RT04(10.184.252.1) — RT01(10.184.252.2)  [10.184.252.0/30]
