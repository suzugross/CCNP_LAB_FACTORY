# 問題 GEN-TWIST-46822 : OSPF ルートフィルタ（自動生成・tree・5台）

各ルータ間リンクと Loopback には IP 設定済み。

## 要件
1. OSPF プロセス 1 / エリア 0 を全ルータ・全リンク・全 Loopback で構成し、すべてのルータが他の全ルータの Loopback に到達できるようにせよ。
2. ただし **RT04** は **RT01 の Loopback (`31.31.31.31/32`)** を自身のルーティングテーブルに**保持しないこと**。
   - 他の全ルータは `31.31.31.31/32` へ到達できること。
   - RT04 自身も、RT01 以外の全 Loopback へは到達できること。

## ループバック

- RT02: 73.73.73.73/32
